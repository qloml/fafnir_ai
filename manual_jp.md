# Fafnir オンライン対戦・サーバー利用マニュアル

このドキュメントは、Fafnirのオンライン対戦を行うためのサーバー (`server_0424.py` / `fast_server_0424.py`) および、AI以外の人間用クライアントや観戦・テスト用ツールの利用方法をまとめたマニュアルです。

---

## 1. サーバー起動

オンライン対戦やGUIツールを使用する際は、まずサーバーを立ち上げる必要があります。
サーバーは `uvicorn` を使用して起動します。

### 通常版サーバー
バグ修正済みの標準サーバーです。
```bash
uvicorn server_0424:socket_app --host 0.0.0.0 --port 8765
```

### 高速版サーバー
アニメーション待機時間などを短縮した高速版サーバーです。
```bash
uvicorn fast_server_0424:socket_app --host 0.0.0.0 --port 8765
```

---

## 2. クライアント・ツール関連 (`clients/` フォルダ)

サーバーを立ち上げた状態で、プレイヤーや観戦者として接続するためのプログラム群です。
（※学習AIボットについては、`mppo_ai` や `cfr_ai` など、各AIフォルダ内の専用マニュアルを参照してください）

### ① `clients/human_cli.py` (CUI人間用クライアント)
ターミナル上で人間がプレイするためのテキストベースのクライアントです。
```bash
python clients/human_cli.py --name Player1 --room room1
```
| 引数 | デフォルト | 説明 |
| :--- | :--- | :--- |
| `--url` | `http://127.0.0.1:8765` | 接続先のサーバーURL。 |
| `--room` | `room1` | 入室するルーム名。 |
| `--name` | `Player` | 画面に表示されるプレイヤーの名前。 |

### ② `clients/ai_bot_sample.py` (ルールベースAIクライアント)
ランダムや簡単なルールで動くサンプルボットです。サーバーの動作確認や、他のAIの対戦相手としてテスト起動するのに便利です。
```bash
python clients/ai_bot_sample.py --name DummyBot --room room1
```
| 引数 | デフォルト | 説明 |
| :--- | :--- | :--- |
| `--url` | `http://127.0.0.1:8765` | 接続先のサーバーURL。 |
| `--room` | `room1` | 入室するルーム名。 |
| `--name` | `Bot` | 画面に表示されるAIの名前。 |

### ③ `clients/web_gui/` (ブラウザ用UI)
ブラウザ上で人間がグラフィカルにプレイするためのWebフロントエンド一式（HTML/CSS/JS）です。
`clients/web_gui/index.html` をブラウザで開くことで、サーバーに接続してプレイできます。特別なコマンドライン起動は不要です。
